# Fanteathy's Blog

## work flow

### new post

`hexo new 'post_title'`

After this, you'll find a new post in `./source/posts` OR you can create `post_title.md` in `./source/posts` directory.

### new page

`hexo new 'page_name'`

### Deploy

`make deploy`
